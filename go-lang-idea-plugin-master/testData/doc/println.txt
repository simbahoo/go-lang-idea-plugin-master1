<b>func Println(a ...interface{}) (n <a href="psi_element://builtin#builtin.int">int</a>, err <a href="psi_element://builtin#builtin.error">error</a>)</b>
<p>Println formats using the default formats for its operands and writes to standard output.
Spaces are always added between operands and a newline is appended.
It returns the number of bytes written and any write error encountered.
</p>
